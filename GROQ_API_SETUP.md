# Настройка Groq API для транскрипции аудио

Groq предоставляет **бесплатный** доступ к Whisper Large V3 для транскрипции аудио.

## Как получить API ключ:

1. Перейди на https://console.groq.com
2. Зарегистрируйся или войди (можно через Google/GitHub)
3. Перейди в раздел "API Keys"
4. Нажми "Create API Key"
5. Скопируй ключ

## Добавь ключ в .env:

```bash
GROQ_API_KEY=gsk_your_api_key_here
```

## Лимиты (бесплатно):

- **14,400 запросов в день**
- **30 запросов в минуту**
- Модель: `whisper-large-v3`
- Поддержка русского языка: ✅

## Если не добавить ключ:

Бот будет работать, но:
- Транскрипция аудио будет пропущена
- Анализ видео будет только по кадрам (без учёта голоса)

## Альтернативы:

Если не хочешь использовать Groq, можно:
1. Использовать OpenAI Whisper API (платно)
2. Использовать Yandex SpeechKit (платно)
3. Запустить локальный Whisper (требует GPU)
